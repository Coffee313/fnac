## Architecture

```
Network Devices (Switches, APs)
         │
         ▼ (RADIUS on UDP 1812)
    FreeRADIUS Server
         │
         ├─ Reads: clients.conf (auto-generated by FNAC)
         ├─ Reads: mab_users (auto-generated by FNAC)
         └─ Sends: Access-Accept/Reject
         
    FNAC Web UI (HTTP 5000)
         │
         ├─ Device Management
         ├─ Client Management (MAC addresses)
         ├─ Policy Management (VLAN assignments)
         └─ Log Viewer
```

## How It Works

1. **User configures device/client/policy in FNAC UI**
2. **FNAC updates its database**
3. **Config generator creates FreeRADIUS files**
4. **FreeRADIUS reloads configuration**
5. **Network device sends RADIUS request**
6. **FreeRADIUS authenticates using FNAC-generated config**
7. **FNAC logs the authentication attempt**

## Quick Start

### Step 1: Install FreeRADIUS
```bash
sudo apt-get install freeradius freeradius-utils
sudo systemctl start freeradius
```

### Step 2: Deploy FNAC
```bash
git clone https://github.com/YOUR_USERNAME/fnac.git
cd fnac
./quickstart.sh setup
sudo -E ./quickstart.sh run
```

### Step 3: Configure via Web UI
- Open `http://server-ip:5000`
- Add device (switch/AP)
- Add client (MAC address)
- Create policy (VLAN assignment)

### Step 4: Test
```bash
radtest -t mschap 00:11:22:33:44:55 mab 127.0.0.1 1812 shared_secret
```
