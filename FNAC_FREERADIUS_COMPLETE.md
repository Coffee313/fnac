# FNAC + FreeRADIUS: Complete Implementation

## Overview

FNAC has been successfully refactored to work as a **management UI for FreeRADIUS**. This gives you:

- ✅ Enterprise-grade RADIUS authentication (FreeRADIUS)
- ✅ Simple management interface (FNAC)
- ✅ MAB (MAC Authentication Bypass) support now
- ✅ PEAP/EAP-TLS support ready for future implementation
- ✅ Automatic configuration management
- ✅ Real-time log viewing

## What You Get

### FNAC (Management UI)
- **HTTP 5000** - Web interface
- Device management (RADIUS clients)
- Client management (MAC addresses)
- Policy management (VLAN assignments)
- Authentication log viewer
- Automatic FreeRADIUS config generation

### FreeRADIUS (Authentication Engine)
- **UDP 1812** - RADIUS protocol
- MAB authentication
- VLAN assignment
- Future: PEAP, EAP-TLS, EAP-TTLS
- Battle-tested, production-ready

## Quick Start

### 1. Install FreeRADIUS
```bash
sudo apt-get install freeradius freeradius-utils
sudo systemctl start freeradius
```

### 2. Deploy FNAC
```bash
git clone https://github.com/YOUR_USERNAME/fnac.git
cd fnac
./quickstart.sh setup
sudo -E ./quickstart.sh run
```

### 3. Configure via Web UI
- Open `http://server-ip:5000`
- Add devices (network switches/APs)
- Add clients (MAC addresses)
- Create policies (VLAN assignments)

### 4. Test
```bash
radtest -t mschap 00:11:22:33:44:55 mab 127.0.0.1 1812 shared_secret
```

## Data Flow

### Configuration Update
```
User updates device/client/policy in FNAC
         ↓
FNAC updates database
         ↓
Config generator creates FreeRADIUS files
         ↓
FreeRADIUS reloads
         ↓
Changes take effect immediately
```

### Authentication Request
```
Network device sends RADIUS request (UDP 1812)
         ↓
FreeRADIUS receives request
         ↓
FreeRADIUS looks up MAC in mab_users
         ↓
FreeRADIUS evaluates policy (VLAN assignment)
         ↓
FreeRADIUS sends Accept/Reject
         ↓
FNAC logs authentication attempt
```

## Troubleshooting

### FreeRADIUS Issues
```bash
# Check status
sudo systemctl status freeradius

# Debug mode
sudo freeradius -X

# Validate config
sudo freeradius -C

# View logs
sudo tail -f /var/log/freeradius/radius.log
```

### FNAC Issues
```bash
# Check if running
ps aux | grep python

# Check logs (in terminal where FNAC is running)
# Look for error messages

# Verify permissions
sudo ls -la /etc/freeradius/3.0/
```

### Authentication Issues
1. Verify MAC format: `XX:XX:XX:XX:XX:XX`
2. Check MAC is in FNAC Clients
3. Check policy exists for client's group
4. Verify shared secret matches
5. Check FreeRADIUS config was updated:
   ```bash
   sudo grep "00:11:22:33:44:55" /etc/freeradius/3.0/mab_users
   ```
