# FNAC: RADIUS Management UI for FreeRADIUS

**FNAC** is a web-based management interface for **FreeRADIUS**, providing simple configuration and monitoring of RADIUS authentication.

## Features

- 🔐 **MAB Authentication** - Authenticate devices by MAC address
- 🏢 **Enterprise-Grade** - Powered by FreeRADIUS
- 🎯 **VLAN Assignment** - Automatically assign VLANs to authenticated devices
- 📊 **Real-Time Logs** - Monitor authentication attempts
- ⚙️ **Auto-Config** - Automatically generates FreeRADIUS configuration
- 🚀 **Future-Ready** - Ready for PEAP/EAP-TLS support

## Quick Start

### 1. Install FreeRADIUS
```bash
sudo apt-get install freeradius freeradius-utils
sudo systemctl start freeradius
```

### 2. Deploy FNAC
```bash
git clone https://github.com/YOUR_USERNAME/fnac.git
cd fnac
./quickstart.sh setup
sudo -E ./quickstart.sh run
```

### 3. Open Web UI
```
http://server-ip:5000
```

### 4. Configure
- Add device (switch/AP)
- Add client (MAC address)
- Create policy (VLAN assignment)

### 5. Test
```bash
radtest -t mschap 00:11:22:33:44:55 mab 127.0.0.1 1812 shared_secret
```

## API Endpoints

### Devices
- `GET /api/devices` - List all devices
- `POST /api/devices` - Create device
- `DELETE /api/devices/<id>` - Delete device

### Clients (MAC Addresses)
- `GET /api/clients` - List all clients
- `POST /api/clients` - Create client
- `DELETE /api/clients/<mac>` - Delete client

### Policies
- `GET /api/policies` - List all policies
- `POST /api/policies` - Create policy
- `DELETE /api/policies/<id>` - Delete policy

### Logs
- `GET /api/logs` - View authentication logs
- Filter by MAC address or outcome

## Configuration

### Device Configuration
```json
{
  "id": "switch_1",
  "ip_address": "192.168.1.1",
  "shared_secret": "your_secret",
  "device_group_id": "switches"
}
```

### Client Configuration
```json
{
  "mac_address": "00:11:22:33:44:55",
  "client_group_id": "printers"
}
```

### Policy Configuration
```json
{
  "id": "printer_policy",
  "client_group_id": "printers",
  "decision": "accept_with_vlan",
  "vlan_id": 100
}
```

## Troubleshooting

### FreeRADIUS not responding
```bash
sudo systemctl status freeradius
sudo freeradius -X  # Debug mode
```

### Config not updating
```bash
# Check permissions
sudo ls -la /etc/freeradius/3.0/

# Check FNAC logs (in terminal)
```

### MAC not authenticating
1. Verify MAC format: `XX:XX:XX:XX:XX:XX`
2. Check MAC is in FNAC Clients
3. Check policy exists for client's group
4. Verify shared secret matches
