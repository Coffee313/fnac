## Technical Details

### Architecture

```
Network Devices (Switches, APs)
         │
         ▼ (UDP 1812)
    FreeRADIUS
         │
         ├─ clients.conf (auto-generated)
         ├─ mab_users (auto-generated)
         └─ Handles RADIUS protocol
         
    FNAC UI (HTTP 5000)
         │
         ├─ Device Management
         ├─ Client Management
         ├─ Policy Management
         └─ Log Viewer
```

### Data Flow

1. User configures device/client/policy in FNAC UI
2. FNAC updates database
3. Config generator creates FreeRADIUS files
4. FreeRADIUS reloads
5. Network device sends RADIUS request
6. FreeRADIUS authenticates using FNAC config
7. FNAC logs authentication attempt

### Configuration Files (Auto-Generated)

**`/etc/freeradius/3.0/clients.conf`**:
```
# Auto-generated by FNAC
client 192.168.1.1 {
    secret = "shared_secret"
    shortname = "device_id"
    nastype = generic
}
```

**`/etc/freeradius/3.0/mab_users`**:
```
# Auto-generated by FNAC
00:11:22:33:44:55 Cleartext-Password := "mab"
    Tunnel-Type = VLAN,
    Tunnel-Medium-Type = IEEE-802,
    Tunnel-Private-Group-ID = "100"
```

## Deployment Path

1. **Install FreeRADIUS**
   ```bash
   sudo apt-get install freeradius freeradius-utils
   sudo systemctl start freeradius
   ```

2. **Deploy FNAC**
   ```bash
   git clone https://github.com/YOUR_USERNAME/fnac.git
   cd fnac
   ./quickstart.sh setup
   sudo -E ./quickstart.sh run
   ```

3. **Configure via Web UI**
   - Open `http://server-ip:5000`
   - Add devices, clients, policies

4. **Test**
   ```bash
   radtest -t mschap 00:11:22:33:44:55 mab 127.0.0.1 1812 secret
   ```
