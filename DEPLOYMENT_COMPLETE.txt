QUICK START (COPY & PASTE):
═══════════════════════════════════════════════════════════════════════════

On your Astra Linux VM:

  # 1. Prepare system
  sudo apt update && sudo apt upgrade -y
  sudo apt install -y python3 python3-pip python3-venv curl

  # 2. Setup server
  cd /path/to/simple-radius-server
  ./quickstart.sh setup

  # 3. Start server
  ./quickstart.sh run

  # 4. In another terminal, configure first device
  curl -X POST http://localhost:5000/api/device-groups \
    -H "Content-Type: application/json" \
    -d '{"id": "switches", "name": "Network Switches"}'

  curl -X POST http://localhost:5000/api/devices \
    -H "Content-Type: application/json" \
    -d '{
      "id": "switch-01",
      "ip_address": "192.168.1.100",
      "shared_secret": "my-secret-key-12345",
      "device_group_id": "switches"
    }'

  # 5. Open web UI
  http://localhost:5000

SUPPORT & TROUBLESHOOTING:
═══════════════════════════════════════════════════════════════════════════

Common Issues:

  "Permission denied" on port 1812
  → Run with sudo: sudo -E ./quickstart.sh run

  "Address already in use"
  → sudo lsof -i :1812
  → sudo kill -9 <PID>

  "ModuleNotFoundError"
  → source venv/bin/activate

  "Can't connect to web UI"
  → ps aux | grep python3 | grep src.main
